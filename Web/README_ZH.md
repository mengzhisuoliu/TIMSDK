## 关于

Tencent Chat SDK 每日承载数十亿条消息，植根于腾讯生态。接入预置 **UIKit**，10 分钟内上线完整聊天体验；或选用 **Core SDK**，在经过大规模验证的消息基础设施之上构建专属 UI。
